Interaktive Visualisierung zur Analyse von Intensivbettenkapazitäten für Covid-19-Patienten

Bitte erlauben Sie einige Sekunden, bis die Kapazitätsdarstellungen geladen sind. Das Tool funktioniert am Besten im Chrome-Browser.

Die Arbeitsgruppe Datenanalyse und Visualisierung an der Universität Konstanz entwickelt dieses Tool, um eine intuitive Übersicht über freie und belegte Bettenkapazitäten in Deutschland zu zeigen. Anhand dieser Information können Engpässe früh erkannt und eine Umverteilung oder direkte Umleitung von Patientenströmen vorgenommen werden.

Die derzeit integrierten Daten umfassen die öffentlichen Bettenauslastungsdaten der DIVI[1] (Link). Durch einen Zugriff auf die exakten Kapazitätsdaten der DIVI könnte unser Tool um nützliche Funktionalitäten, insbesondere die Frühwarnung vor Überlastungen erweitert werden. Diese Funktionalitäten sind schon vorhanden, jedoch aufgrund fehlendem Datenzugang derzeit deaktiviert.

Das Tool ist kurzfristig einsatzfähig und nicht nur ein Forschungsprototyp. Es wird mit Experten sinnvoll weiterentwickelt, um bestmögliche Unterstützung leisten zu können. Einen kurzen Videoüberblick finden Sie unter https://video.coronavis.dbvis.de.

Auf den folgenden Seiten stellen wir Ihnen die Funktionalitäten des Tools vor. Wenn Sie direkt das Tool anschauen wollen, kann dieses Fenster geschlossen werden.
